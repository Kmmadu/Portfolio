## ✨ Features

- 🎨 **Modern Dark Theme** - Tech-inspired aesthetic with network nodes and abstract patterns
- 📱 **Fully Responsive** - Mobile-first design that works on all devices
- ⚡ **Smooth Animations** - Scroll-based reveals and interactive elements
- 🧠 **Particle Background** - Dynamic network-style animation
- 📧 **Working Contact Form** - Integrated with Formspree for real message delivery
- 🔍 **SEO Friendly** - Semantic HTML and optimized meta tags
- 🎯 **Interactive UI** - Active section highlighting, typing effects, and number counters

## 🖥️ Local Development

### Prerequisites
- Any modern web browser
- Local server (optional - works with direct file opening)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Kmmadu/kingsley-portfolio.git
   cd kingsley-portfolio
   
2. **Open the website**
   ```bash
   # Using Python (optional)
   python -m http.server 8000
   # Or simply open index.html in your browser
    open index.html

3. **Customise for your needs**
   
- Update profile photo in assets/ folder
- Modify content in index.html
- Adjust colours in: root variables in style.css
- Update Formspree endpoint in contact form
