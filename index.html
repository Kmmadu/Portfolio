Resolved

Complete Office Network Outage

Problem Statement

Client reported complete internet outage across the office network. Physical connectivity was confirmed, but users were unable to access online services or internal resources.

Investigation Process
  • Traced PoE cable path from router to wireless radio
  • Performed direct laptop connection tests to isolate failure point
  • Accessed MikroTik router through WinBox for configuration audit
  • Identified missing VLAN assignments on the main bridge interface after power interruption
  • Verified broken network segmentation and DHCP relay communication
Resolution Applied

Restored required VLAN configurations through WinBox, verified DHCP relay functionality, and confirmed successful internet connectivity across the client environment.

WinBox Laptop Diagnostics VLAN Configuration

Resolved

Slow Internet Link Performance Investigation

Problem Statement

Client reported slow internet performance. The objective was to determine whether the issue was caused by wireless connectivity, network configuration, or bandwidth utilization.

Investigation Process
  • Connected directly to client network equipment through PoE for isolation testing
  • Configured static IP settings including IP address, subnet mask, gateway, and DNS
  • Verified internet access after network configuration
  • Performed bandwidth testing using multiple speed test servers
  • Compared results based on server location, routing path, and performance
  • Used MikroTik WinBox to review wireless link performance between base radio and customer radio
Findings & Resolution

Testing confirmed that the wireless link was operating normally. Investigation suggested that the customer was reaching their subscribed bandwidth capacity. The recommended solution was bandwidth upgrade rather than infrastructure repair.

Key Learning

Learned the importance of structured troubleshooting. A single speed test does not always represent complete network performance. Testing with different servers provides better visibility into routing, latency, and bandwidth behaviour.

MikroTik Network Diagnostics Bandwidth Testing WinBox
